======================================================================
PostgreSQL + pgAdmin Setup on Ubuntu VM (Manual Installation)
======================================================================

======================================================================
STEP 0: Connect to the Linux VM from Your Local Machine
======================================================================

If you're using Azure, AWS, or GCP, connect using SSH:

    ssh <your-username>@<your-vm-public-ip>

Example:

    ssh opensi@20.50.120.100

If you're using a private key:

    ssh -i path/to/key.pem opensi@<your-vm-public-ip>

You are now inside the VM terminal.

======================================================================
STEP 1: Install PostgreSQL
======================================================================

    sudo apt update
    sudo apt install postgresql postgresql-contrib

======================================================================
STEP 2: Add 'postgres' User to SUDO Group
======================================================================

    sudo usermod -aG sudo postgres

======================================================================
STEP 3: Set a Linux Password for the 'postgres' User
======================================================================

    sudo passwd postgres

======================================================================
STEP 4: Set a PostgreSQL Account Password
======================================================================

    sudo -u postgres psql -c "ALTER USER postgres PASSWORD 'YourSecurePassword';"

======================================================================
STEP 5: Access PostgreSQL Shell
======================================================================

    sudo su -l postgres
    psql

======================================================================
STEP 6: Create a Test Database and User
======================================================================

Inside the psql shell:

    CREATE DATABASE mytestdb;
    CREATE USER mytestuser WITH ENCRYPTED PASSWORD 'MyStr0ngP@SS';
    GRANT ALL PRIVILEGES ON DATABASE mytestdb TO mytestuser;

List databases:

    \l

Connect to your new DB:

    \c mytestdb

======================================================================
STEP 7: Enable Remote Connections
======================================================================

1. Edit postgresql.conf:

    sudo nano /etc/postgresql/<version>/main/postgresql.conf

Uncomment and update this line:

    listen_addresses = '*'

2. Edit pg_hba.conf:

    sudo nano /etc/postgresql/<version>/main/pg_hba.conf

Add:

    host all all 0.0.0.0/0 md5

(Replace with subnet CIDR for production)

======================================================================
STEP 8: Restart PostgreSQL
======================================================================

    sudo systemctl restart postgresql

======================================================================
STEP 9: Install and Configure pgAdmin Web
======================================================================

    sudo apt install pgadmin4-web
    sudo /usr/pgadmin4/bin/setup-web.sh

Enter admin email and password.

======================================================================
STEP 10: Allow Web Access from Local Machine
======================================================================
On the local machine terminal:

    ssh -L <local-port>:localhost:<remote-port> <user>@<vm-ip>

======================================================================
Step 11: Create Simple Docker that Access to Postgres
======================================================================
    docker run -d -p 8080:8080 \
    -e OPENAI_API_KEY=your_key \
    -e WEBUI_SECRET_KEY=your_secret \
    -e DATABASE_URL=postgresql://opeuser:data101@10.0.0.4:5432/opeweb \
    --name openwebui \

Access the openweb from port 8080

Access the pgadmin from http://127.0.0.1:5050/pgadmin4/
